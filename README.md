# typebox-codegen-deno-doc

Code generation for [TypeBox](https://github.com/sinclairzx81/typebox) types
from the [Deno Doc API](https://jsr.io/@deno/doc).

## Develop

Run the tests:

```bash
deno task test
```

Generate code:

```bash
deno task generate
```

Update outdated dependencies:

```bash
deno task outdated
```
