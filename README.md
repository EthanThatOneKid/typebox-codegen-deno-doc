# typebox-codegen-deno-doc

Code generation for TypeBox types from Deno Doc API.

## Develop

Run the tests:

```bash
deno task test
```

Generate code:

```bash
deno task generate
```

Update outdated dependencies:

```bash
deno task outdated
```
